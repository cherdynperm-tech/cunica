# Cunica Memory Bank

Это постоянная база контекста для продолжения разработки `cunica` в новых сессиях.

Alias-термины проекта для поиска: `1с`, `1c`, `bsl`, `cursor`, `unica`.

## Быстрый старт (5 минут)

1. Прочитать [project-overview.md](./project-overview.md).
2. Прочитать [architecture.md](./architecture.md).
3. Проверить команды в [commands-runbook.md](./commands-runbook.md).
4. Проверить текущий контракт в [unica-contract.md](./unica-contract.md).
5. Открыть [next-steps.md](./next-steps.md) и выбрать следующую задачу.
